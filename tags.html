Tercer Reto

secciones

  1. La etiqueta HEADER: Se encuentra en la parte superior de la página donde puede ir un logo, un elemento de encabezado, formulario de búsqueda, un nombre del autor, etc.
  2. La etiqueta SECTION: Define una sección en un documento, en HTML 5 remplaza a la etiqueta div,
  3. La etiqueta NAV: Son enlaces de navegación, pueden ser menús, tabla de contenido o índices
  4. La etiqueta ARTICLE: Define contenido autónomo que podría existir independientemente del resto del contenido, por ejemplo: mensaje de un foro, un artículo de una revista o periódico, una entrada de un blog, un comentario de un usuario, un widget interactivo o gadget, etc.
  5. La etiqueta ASIDE: Esta indirectamente relacionado al documento, por ejemplo: barras laterales; publicidad, biografía del autor, enlaces de blogs relacionados, etc.
  6. La etiqueta FOOTER: Normalmente se encuentra en la parte inferior de la página, también un el footer puede estar dentro de un "
    ", definiendo el pie de una página o sección
  7. Entre otros

Agrupación de Contenido

  1. La etiqueta P: Para crear parrafos
  2. La etiqueta OL: Para crear lista numericos, entre otros
  3. La etiqueta UL: Para crear lista por simbolos, alfabeticos, etc.
  4. Entre otros

Contenido incrustado

  1. La etiqueta IMG: Para inscrustar una imagen y usas la propiedad src y alt, entre otras.
  2. La etiqueta EMBED: Para inscrustar un recurso externo, como un video, imagen, etc.
  3. La etiqueta AUDIO: Representa un sonido o stream de audio.
  4. Entre otros
